# Tarif Kartı (HTML Alıştırma)

Bu proje, **öğrenme amaçlı** hazırlanmış basit bir HTML tarif kartıdır.

## Amaç
- Temel HTML yapısını öğrenmek
- Liste kullanımı (`ul`, `ol`) pratiği yapmak
- Başlıklar, paragraflar ve yorum satırlarını anlamak
- Coddy üzerinden adım adım ilerlenen dersleri uygulamak

## Kullanılan Teknolojiler
- HTML5

## Notlar
Bu proje **başlangıç seviyesi** bir alıştırmadır.  
Bilerek CSS veya JavaScript eklenmemiştir.
