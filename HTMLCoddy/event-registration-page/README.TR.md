# Etkinlik Kayıt Sayfası (HTML Pratiği)

Bu proje, öğrenme amaçlı oluşturulmuş basit bir HTML etkinlik kayıt sayfasıdır.

## Amaç

- Temel HTML yapısını pratik yapmak  
- Tablo ve form kullanımını öğrenmek  
- Başlıklar, paragraflar, resimler ve bağlantıları anlamak  
- Coddy üzerinden adım adım dersleri takip etmek

## Kullanılan Teknolojiler

- HTML5

## Notlar

- Bu proje başlangıç seviyesinde bir uygulamadır.  
- CSS veya JavaScript bilinçli olarak eklenmemiştir.
