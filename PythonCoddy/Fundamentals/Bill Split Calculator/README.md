# Bill Split Calculator

This is a simple Python project that calculates the total bill including tip and splits it among a number of people.

## How it works
Enter the bill amount.
Enter the tip percentage.
Enter the number of people to split the bill.

## The program calculates:
Total bill including tip
Each person's share

## Example

**Input:**
Enter the bill amount: 100
Enter the tip percentage: 5
Enter the number of people to split the bill: 2


**Output:**
Bill Split Calculator
Total (including tip): $105.0
Each person pays: $52.5

## Notes
Developed step by step using Coddy.
Input prompts are in English.
Floats are used for calculations; no rounding is applied.
